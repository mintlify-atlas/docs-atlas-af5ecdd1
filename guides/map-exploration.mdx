## MapLibre Integration

### Setting Up the Map

The map is integrated using AndroidView to embed the native MapView:

```kotlin
AndroidView(
    modifier = Modifier
        .fillMaxSize()
        .padding(padding),
    factory = { context ->
        MapLibre.getInstance(context)
        val mapView = MapView(context)
        mapView.onCreate(null)
        mapView.getMapAsync { map ->
            map.setStyle(
                Style.Builder().fromUri("https://tiles.openfreemap.org/styles/liberty")
            ) {
                // Map configuration
            }
        }
        mapView
    },
    update = { mapView ->
        mapView.onResume()
    }
)
```

Key components:
- `MapLibre.getInstance(context)` initializes the library
- `getMapAsync` ensures the map is ready before configuration
- `Style.Builder().fromUri()` loads the OpenFreeMap Liberty style
- `update` lambda handles lifecycle events

<Note>
  The "liberty" style provides a clean, readable map design suitable for showing cultural and geographic information.
</Note>

## Adding Story Markers

Stories with latitude and longitude data are displayed as markers:

```kotlin
storiesWithLocation.forEach { story ->
    map.addMarker(
        MarkerOptions()
            .position(LatLng(story.latitude!!, story.longitude!!))
            .title(story.titleEs)
            .snippet("Narrado por ${story.narratorName}")
    )
}
```

Each marker includes:
- **Position**: The story's latitude/longitude coordinates
- **Title**: Spanish title of the legend
- **Snippet**: "Narrado por [narrator name]" for context

<Tip>
  When multiple stories share similar locations, markers may overlap. Zoom in to see individual markers more clearly.
</Tip>

## Marker Interaction

### Tap to Preview

When a user taps a marker, a bottom sheet appears:

```kotlin
var selectedStory by remember { mutableStateOf<Story?>(null) }
var showSheet by remember { mutableStateOf(false) }

map.setOnMarkerClickListener { marker ->
    selectedStory = storiesWithLocation.find { it.titleEs == marker.title }
    if (selectedStory != null) {
        showSheet = true
    }
    true
}
```

<Steps>
  <Step title="Tap Marker">
    User taps any marker on the map.
  </Step>
  
  <Step title="Find Story">
    The click listener matches the marker title to find the corresponding story object.
  </Step>
  
  <Step title="Show Preview">
    A bottom sheet slides up with story details.
  </Step>
  
  <Step title="Navigate or Dismiss">
    User can read the full story or dismiss to continue exploring the map.
  </Step>
</Steps>

### Bottom Sheet Preview

The bottom sheet displays key story information:

```kotlin
if (showSheet && selectedStory != null) {
    ModalBottomSheet(
        onDismissRequest = { showSheet = false },
        containerColor = FondoTarjeta,
        dragHandle = { BottomSheetDefaults.DragHandle(color = AguaSagrada) }
    ) {
        Column(
            modifier = Modifier
                .fillMaxWidth()
                .padding(24.dp),
            verticalArrangement = Arrangement.spacedBy(16.dp)
        ) {
            Text(
                text = selectedStory!!.titleEs,
                style = MaterialTheme.typography.headlineSmall,
                fontWeight = FontWeight.Bold,
                color = LunaLlena
            )
            
            if (selectedStory!!.titleNahuatl.isNotBlank()) {
                Text(
                    text = selectedStory!!.titleNahuatl,
                    style = MaterialTheme.typography.titleMedium,
                    fontStyle = FontStyle.Italic,
                    color = AguaSagrada
                )
            }
            
            Text(
                text = selectedStory!!.contentEs,
                style = MaterialTheme.typography.bodyMedium,
                color = TextoSecundario,
                maxLines = 3,
                overflow = TextOverflow.Ellipsis
            )
            
            Button(
                onClick = { 
                    showSheet = false
                    onStoryClick(selectedStory!!.id) 
                },
                modifier = Modifier.fillMaxWidth().height(50.dp),
                colors = ButtonDefaults.buttonColors(
                    containerColor = AguaSagrada,
                    contentColor = NocheProfunda
                ),
                shape = RoundedCornerShape(12.dp)
            ) {
                Text("Leer leyenda completa", fontWeight = FontWeight.Bold)
            }
        }
    }
}
```

The bottom sheet includes:
1. **Spanish Title**: Main heading in bold
2. **Nahuatl Title**: Italic turquoise text (if available)
3. **Content Preview**: First 3 lines of the story with ellipsis
4. **Action Button**: "Leer leyenda completa" to open full details

<Warning>
  The content preview is truncated to 3 lines to keep the bottom sheet compact. Users must tap the button to read the complete story.
</Warning>

## Technical Details

### MapLibre vs Google Maps

Casa de Historias uses MapLibre GL with OpenFreeMap because:

- **Open Source**: No API keys or usage limits
- **Free Tiles**: OpenFreeMap provides tiles at no cost
- **Customizable**: Full control over map styling
- **Privacy**: No tracking or data collection
- **Offline Support**: Tiles can be cached for offline use

### Coordinate System

Locations use standard geographic coordinates:
- **Latitude**: -90 to 90 degrees (N/S)
- **Longitude**: -180 to 180 degrees (E/W)
- Sierra de Zongolica: ~18.6667°N, 97.0000°W

### Performance Considerations

The map handles large numbers of stories efficiently:

```kotlin
val storiesWithLocation = stories.filter { 
    it.latitude != null && it.longitude != null 
}
```

- Only stories with valid coordinates are processed
- Markers are added in a single pass during map initialization
- MapLibre clusters markers at low zoom levels automatically

<Info>
  As the story collection grows, MapLibre's built-in marker clustering ensures the map remains performant and readable at all zoom levels.
</Info>
