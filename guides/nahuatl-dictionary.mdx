## Search Functionality

### Real-time Filtering

The search field filters words as you type:

```kotlin
var searchQuery by remember { mutableStateOf("") }

val dictionary = listOf(
    NahuatlWord("Tonatiuh", "Sol", "El dador de vida."),
    NahuatlWord("Metztli", "Luna", "La que brilla en la noche mística."),
    NahuatlWord("Tlalticpac", "Tierra / Mundo", "Sobre la tierra donde caminamos."),
    // ... more words
)

val filteredDictionary = dictionary.filter {
    it.nahuatl.contains(searchQuery, ignoreCase = true) || 
    it.spanish.contains(searchQuery, ignoreCase = true)
}.sortedBy { it.nahuatl }
```

Key features:
- **Case-insensitive**: Searches work regardless of capitalization
- **Bilingual**: Matches in both Nahuatl and Spanish fields
- **Sorted**: Results are alphabetically sorted by Nahuatl word
- **Reactive**: Updates instantly as the user types

## Word Data Model

Each dictionary entry uses a simple data class:

```kotlin
data class NahuatlWord(
    val nahuatl: String,
    val spanish: String,
    val example: String = ""
)
```

**Fields:**
- `nahuatl`: The indigenous word
- `spanish`: Spanish translation
- `example`: Optional usage example or cultural context

## Word Card Component

Each word displays in a styled card:

```kotlin
@Composable
fun WordCard(word: NahuatlWord) {
    Card(
        modifier = Modifier.fillMaxWidth(),
        shape = RoundedCornerShape(16.dp),
        colors = CardDefaults.cardColors(containerColor = FondoTarjeta),
        border = BorderStroke(
            1.dp, 
            Color.White.copy(alpha = 0.05f)
        )
    ) {
        Column(
            modifier = Modifier
                .fillMaxWidth()
                .padding(16.dp)
        ) {
            Row(
                modifier = Modifier.fillMaxWidth(),
                horizontalArrangement = Arrangement.SpaceBetween,
                verticalAlignment = Alignment.CenterVertically
            ) {
                Text(
                    text = word.nahuatl,
                    style = MaterialTheme.typography.titleLarge,
                    color = AguaSagrada,
                    fontStyle = FontStyle.Italic,
                    fontWeight = FontWeight.Bold
                )
                Text(
                    text = word.spanish,
                    style = MaterialTheme.typography.titleMedium,
                    color = LunaLlena,
                    fontWeight = FontWeight.Medium
                )
            }
            if (word.example.isNotBlank()) {
                Spacer(modifier = Modifier.height(8.dp))
                Box(
                    modifier = Modifier
                        .fillMaxWidth()
                        .background(SurfaceVariant, RoundedCornerShape(8.dp))
                        .padding(12.dp)
                ) {
                    Text(
                        text = "Ej: ${word.example}",
                        style = MaterialTheme.typography.bodyMedium,
                        color = TextoSecundario,
                        fontStyle = FontStyle.Italic
                    )
                }
            }
        }
    }
}
```

## List Display

Words display in a vertically scrollable list:

```kotlin
LazyColumn(
    modifier = Modifier.fillMaxSize(),
    contentPadding = PaddingValues(bottom = 24.dp),
    verticalArrangement = Arrangement.spacedBy(12.dp)
) {
    items(filteredDictionary) { word ->
        WordCard(word)
    }
}
```

- **Spacing**: 12dp between cards for visual clarity
- **Bottom Padding**: 24dp prevents the last item from being hidden by navigation
- **Reactive**: List updates automatically when search query changes

## Technical Implementation

### Data Flow

```
User Input → searchQuery State
     ↓
Filter Function (nahuatl + spanish fields)
     ↓
Sorted List → filteredDictionary
     ↓
LazyColumn → WordCard Components
```

### Performance

- **Reactive State**: `remember { mutableStateOf("") }` ensures efficient recomposition
- **Simple Filtering**: O(n) filter operation is fast even with hundreds of words
- **Lazy Loading**: `LazyColumn` only renders visible items
- **No Network**: Static list loads instantly without API calls

<Info>
  With the current 10-word dictionary, performance is instantaneous. Even with 1000+ words, the filter and sort operations would remain performant on all devices.
</Info>
