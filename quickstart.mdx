This guide will help you set up Casa de Historias locally and start exploring indigenous legends from the Sierra de Zongolica.

<Note>
**Prerequisites**: You'll need Android Studio, JDK 17, and a Firebase project configured with Firestore, Authentication, and Storage.
</Note>

## Quick Setup

<Steps>
  <Step title="Clone the repository">
    Clone the Casa de Historias repository to your local machine:

    ```bash
    git clone https://github.com/joseorteha/app-leyendas.git
    cd app-leyendas
    ```
  </Step>

  <Step title="Set up Firebase">
    Create a Firebase project and download the `google-services.json` file:

    1. Go to the [Firebase Console](https://console.firebase.google.com)
    2. Create a new project or select an existing one
    3. Add an Android app with package name: `com.example.casadehistorias`
    4. Download `google-services.json`
    5. Place it in the `app/` directory

    <Warning>
    The app will not compile without `google-services.json`. Make sure it's in the correct location: `app/google-services.json`
    </Warning>

    Enable these Firebase services in your console:
    - **Firestore Database** - For storing stories
    - **Authentication** - Email/password provider
    - **Storage** - For audio files and images
  </Step>

  <Step title="Open in Android Studio">
    Open the project in Android Studio:

    1. Launch Android Studio
    2. Select **Open an Existing Project**
    3. Navigate to the cloned `app-leyendas` folder
    4. Click **OK**

    Android Studio will automatically sync Gradle dependencies. This may take a few minutes.
  </Step>

  <Step title="Build and run">
    Build and run the app on an emulator or physical device:

    1. Wait for Gradle sync to complete
    2. Select a target device (emulator or connected phone with Android 8.0+)
    3. Click the **Run** button (green play icon) or press `Shift + F10`

    The app will compile and launch. You'll see the welcome screen with the moon icon and "Casa de Historias" title.
  </Step>

  <Step title="Create an account">
    On first launch, you'll need to create an account:

    1. Tap **Descubrir Historias** on the welcome screen
    2. On the login screen, tap **Crear Cuenta**
    3. Enter your email and password (minimum 6 characters)
    4. Tap **Registrarse**

    You'll be automatically signed in and taken to the Discover screen.
  </Step>

  <Step title="Explore your first story">
    Browse and view stories:

    1. The **Discover** screen shows featured stories
    2. Tap on any story card to view full details
    3. Toggle between Spanish and Nahuatl using the language buttons
    4. Tap the play button to hear audio narration (if available)
    5. Use the heart icon to add stories to your favorites

    <Info>
    If you don't see any stories yet, you'll need to add some through the Firebase Console or by contributing stories through the app's **Add Story** feature.
    </Info>
  </Step>
</Steps>

## Project Structure

Here's what you'll find in the codebase:

```
app-leyendas/
├── app/
│   ├── src/main/java/com/example/casadehistorias/
│   │   ├── data/              # Data layer (repositories, Firebase, Room)
│   │   ├── domain/            # Domain models (Story, etc.)
│   │   ├── ui/                # Jetpack Compose UI screens
│   │   ├── di/                # Dependency injection modules
│   │   └── CasaDeHistoriasApp.kt
│   ├── src/main/res/          # Android resources
│   └── build.gradle.kts       # App-level build configuration
├── build.gradle.kts           # Project-level build configuration
└── settings.gradle.kts
```

## Development Workflow

Once you have the app running, you can:

1. **Run tests**: `./gradlew test` for unit tests
2. **Build APK**: `./gradlew assembleDebug`
3. **View logs**: Use Android Studio's Logcat panel
4. **Hot reload**: Compose supports live updates while running

<Tip>
Enable Kotlin compiler warnings in Android Studio to catch potential issues early. Go to **Settings → Build, Execution, Deployment → Compiler → Kotlin Compiler** and enable all warnings.
</Tip>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Build fails with 'google-services.json not found'">
    Make sure `google-services.json` is placed in the `app/` directory (not the project root). The file should be at:
    
    ```
    app-leyendas/app/google-services.json
    ```

    If the file is missing, download it from your Firebase Console.
  </Accordion>

  <Accordion title="Gradle sync fails">
    Try these steps:
    
    1. **File → Invalidate Caches and Restart**
    2. Delete the `.gradle` folder in your project root
    3. Run `./gradlew clean build` from terminal
    4. Restart Android Studio
  </Accordion>

  <Accordion title="App crashes on launch">
    Check that:
    
    - Your Firebase project has Firestore, Auth, and Storage enabled
    - The package name in `google-services.json` matches `com.example.casadehistorias`
    - You're using Android 8.0 (API 26) or higher
    - Check Logcat for error messages
  </Accordion>

  <Accordion title="No stories appear in the app">
    The Firestore database may be empty. You can:
    
    1. Add stories manually through the Firebase Console (create a `stories` collection)
    2. Use the in-app **Add Story** feature (tap the + button)
    3. Import sample data if you have a JSON export
  </Accordion>
</AccordionGroup>
